# Rumo Certo

Projeto desenvolvido para a disciplina de [web design 1], com objetivo de criar um site institucional para a empresa fictícia **Rumo Certo**, especializada em conserto de eletrodomésticos.

---

##  Funcionalidades

- Página inicial com navegação clara
- Seções: **Serviços**, **Sobre nós**, **Contato**
- Sistema de **Login e Cadastro** integrado ao Firebase Authentication
- Armazenamento de dados de usuários no **Firebase Firestore**
- Deploy realizado no **Firebase Hosting**

---

##  Tecnologias utilizadas

- **HTML5** e **CSS3**
- **JavaScript (ES6+)**
- **Firebase Authentication**
- **Firebase Firestore**
- **Firebase Hosting**
